# Что бы подгрузить изменения с репозитория GitHub
## git pull название_соединения название_ветки
### `git pull origin master`



# Что бы отправить изменения на репозиторий GitHub
## git add название_файла или . (точка означает - добавить все файлы, и он выберет только те, в которых появились новые изменения)
## git commit -m "название_изменения"
## git push название_соединения название_ветки


### `git add file.py`
### `git commit -m "добавил новую функцию registration"`
### `git push origin master`



# Что бы клонировать репозиторий с GitHub (если на нашем компьютере вообще нету этого репозитория)
## git clone адрес
### `git clone https://github.com/user/user_repository`
